# Python + Vue 项目初始化器

这个skill帮助您快速创建基于Python 3.12+FastAPI+PostgreSQL后端和Vue 3+TypeScript前端的完整后台管理系统。

## 标准提示词模板

当用户需要创建此类项目时，使用以下标准提示词：

```markdown
# 后台管理系统项目创建提示词（Python 3.12 + 完整配置）

**请帮我创建一个基于Python 3.12+FastAPI+PostgreSQL后端和Vue 3+TypeScript前端的完整后台管理系统。**

## 项目需求

### 技术栈要求
- **Python版本**: 3.12+
- **后端**: FastAPI + PostgreSQL + SQLAlchemy 2.0
- **前端**: Vue 3 + TypeScript + Element Plus + Vite
- **认证**: JWT Token认证机制

### 数据库配置要求
- **连接信息**: localhost:5432/demo
- **用户名**: frank
- **密码**: 无密码
- **连接池**: 必须配置数据库连接池
- **配置化**: 所有数据库配置通过环境变量或配置文件管理

### 核心功能要求
1. **用户账号管理**（增删改查）
   - 用户列表展示（分页、搜索、筛选）
   - 用户创建（表单验证、密码加密）
   - 用户信息编辑
   - 用户删除（软删除）
   - 用户状态管理（启用/禁用）

2. **登录认证系统**
   - 用户名密码登录
   - JWT Token认证
   - 登录状态保持
   - 安全退出

### 技术要求
1. **数据库配置化**
   - 使用Pydantic Settings管理配置
   - 支持环境变量覆盖
   - 连接池配置（大小、超时等）

2. **日志系统**
   - 标准化的日志打印格式
   - 不同级别的日志记录（DEBUG, INFO, WARNING, ERROR）
   - 结构化日志输出
   - 请求/响应日志记录

3. **异常处理**
   - 统一的异常捕获和处理机制
   - 自定义异常类，每个异常都有错误码
   - 全局异常处理器
   - 统一的错误响应格式

4. **错误码定义**
   - 系统错误码（如：1001-系统错误）
   - 业务错误码（如：2001-用户不存在）
   - 认证错误码（如：3001-令牌过期）

### 数据库要求
- **必须包含完整的建表SQL脚本**
- 使用Alembic进行数据库迁移管理
- 包含用户表、操作日志表等必要表结构
- 包含适当的索引和约束

### 项目结构标准
```
项目名称/
├── backend/
│   ├── app/
│   │   ├── api/           # API路由
│   │   ├── models/        # SQLAlchemy模型
│   │   ├── schemas/       # Pydantic模式
│   │   ├── services/      # 业务逻辑
│   │   ├── core/          # 核心配置
│   │   │   ├── config.py  # 配置管理（包含数据库连接池）
│   │   │   ├── database.py # 数据库连接
│   │   │   └── security.py # 安全配置
│   │   ├── utils/         # 工具函数
│   │   │   ├── logger.py  # 日志配置
│   │   │   └── exceptions.py # 异常定义
│   │   └── middleware/    # 中间件
│   ├── alembic/           # 数据库迁移（包含建表SQL）
│   ├── requirements.txt   # Python 3.12依赖

├── frontend/
│   ├── src/
│   │   ├── views/         # 页面组件
│   │   ├── components/    # 可复用组件

│   └── package.json
└── README.md
```

### 开发标准
1. **Python 3.12特性**
   - 使用新的类型提示语法
   - 利用性能改进特性

2. **配置管理**
   - 使用pydantic-settings管理配置
   - 支持开发/测试/生产环境
   - 敏感信息通过环境变量管理

3. **错误码规范**
   - 系统级错误: 1000-1999
   - 业务级错误: 2000-2999  
   - 认证级错误: 3000-3999
   - 每个错误都有明确的错误码和描述

2. **包含建表SQL的数据库迁移文件**
```
